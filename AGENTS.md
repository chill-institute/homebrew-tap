# Homebrew Tap

`homebrew-tap` publishes Homebrew formulas for `chill.institute` packages.

## Priorities

- Keep formulas aligned with tagged releases from their source repos.
- Prefer small, reviewable formula updates over tap-wide churn.
- Keep install instructions and formula names consistent with the published packages.

## Workflow

- Treat this repo as release plumbing, not product code.
- Update formulas and README guidance together when install behavior changes.
